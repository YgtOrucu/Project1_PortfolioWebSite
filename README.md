# 🌍 Portfolyo Projesi

---

## ✨ Proje Hakkında
Bu proje, **M&Y Yazılım Eğitim Akademi Danışmanlık** bünyesinde  
**Murat Yücedağ**’ın eğitim müfredatında yer alan projelerin birer *case* olarak ele alınmasıyla geliştirilmiştir.

**ASP.NET Core 8.0** altyapısı kullanılarak geliştirilen bu Dinamik Yönetim Panelli Portfolyo Sistemi,  
modern bir web uygulamasının backend ve frontend süreçlerini uçtan uca kapsayan bir full-stack çalışmadır.

---

## 🏗️ Mimari Yaklaşım
- ASP.NET Core 8.0
- Entity Framework Core (Code First)
- Asenkron programlama modeli

---

## 🗄️ Veritabanı Yönetimi
- SQL Server
- Çok ilişkili tablolar (Eğitim, Deneyim, Yetenekler, Projeler vb.)
- Asenkron CRUD işlemleri
- Veri bütünlüğü ve performans odaklı yapı

---

## 🛡️ Hata Yönetimi ve Validation
- SQL kısıtlamalarından (Not Null vb.) kaynaklanan problemler
- Güvenli update metotları
- Model bazlı doğrulama (validation) süreçleri

---

## 🎨 Frontend ve UI
- StitchAI destekli tasarım süreci
- Tailwind CSS ile responsive admin paneli
- CSS optimizasyonları ile pürüzsüz kullanıcı deneyimi

---

## ⚙️ Fonksiyonel Özellikler
- Hakkımda, Yetenekler, Deneyimler, Projeler, Mesajlar modülleri
- Tam kapsamlı CRUD işlemleri
- Ajax ve SweetAlert2 ile sayfa yenilenmeden çalışan yapı
- View Component tabanlı modüler mimari

---

## 🛠️ Kullanılan Teknolojiler
- ASP.NET Core 8.0
- Entity Framework Core
- SQL Server
- Tailwind CSS
- JavaScript
